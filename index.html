Interactive Context-Aware Framework for Fine-Grained 3D Character Facial Editing

Anonymous Authors


Abstract

Text-guided 3D in-game character auto-customization is effective in simplifying the complex process of adjusting numerous character control parameters. However, in dialogue-based editing, existing methods struggle with fine-grained attribute modifications, context-aware editing strength control, and generating results that align with user preferences. To address these challenges, we propose a Context-aware Interactive Character Facial Editing Framework that improves both data curation and model design. Specifically, we establish a standardized data curation pipeline that generates image-text pairs with fine-grained correspondence, i.e.,, character attributes and context-aware editing strength. For precise modifications, we introduce the Attribute and Strength Extraction Model (ASEM) to identify target editing parameters and determine adjustment strength based on both historical and current instructions. Additionally, we develop the Fine-grained Character Parameter Editing (FCPE) to ensure controllable and reliable modifications by generating post-edited character parameters from extracted text prompts and editing strength. To achieve user-preference-aware editing, we introduce Direct Preference Optimization (DPO) into our conditional generative model, enabling it to effectively align with user preferences while requiring minimal user feedback data. Extensive experiments validate the superior editing performance of our proposed framework for in-game character customization.
